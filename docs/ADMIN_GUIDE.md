## 員工管理

### 新增員工

1. 進入「系統設定」→「員工管理」
2. 點擊「新增員工」
3. 填寫員工基本資料：
   - 員工編號（必填，唯一）
   - 姓名
   - 部門
   - 職位
   - 入職日期
   - 底薪
4. 點擊「儲存」

### 建立登入帳號

1. 在員工列表中找到該員工
2. 點擊「建立帳號」
3. 設定：
   - 使用者名稱（預設為員工編號）
   - 初始密碼
   - 角色權限
4. 員工首次登入時會被要求變更密碼

### 編輯員工資料

1. 點擊員工列表中的「編輯」
2. 修改需要變更的資料
3. 點擊「儲存」

### 停用/離職處理

1. 點擊員工的「停用」按鈕
2. 確認停用後，該員工將無法登入

## 考勤管理

### 查看考勤記錄

**路徑：** 「考勤管理」→「考勤記錄」

1. 選擇查詢月份
2. 可按部門或員工篩選
3. 查看每日打卡記錄

### 補登打卡記錄

**路徑：** 「考勤管理」→「補登打卡」

1. 選擇員工
2. 選擇日期
3. 輸入打卡時間
4. 填寫補登原因
5. 點擊「送出」

### 考勤異常處理

**路徑：** 「考勤管理」→「異常記錄」

| 異常類型 | 處理方式 |
|---------|---------|
| 缺勤 | 確認是否為請假或補登 |
| 遲到 | 依公司規定處理 |
| 早退 | 確認是否有申請 |
| 打卡異常 | 確認是否需補登 |

## 請假與加班審核

### 審核請假申請

**路徑：** 「審核管理」→「請假審核」

1. 查看待審核列表
2. 點擊申請查看詳情
3. 確認請假資訊：
   - 請假類型
   - 請假日期
   - 請假天數
   - 請假事由
4. 點擊「核准」或「駁回」
5. 如駁回需填寫原因

### 審核加班申請

**路徑：** 「審核管理」→「加班審核」

1. 查看待審核列表
2. 確認加班資訊：
   - 加班日期
   - 加班時段
   - 加班理由
   - 補償方式
3. 點擊「核准」或「駁回」

## 系統設定

### GPS 打卡設定

**路徑：** 「系統設定」→「GPS打卡設定」

#### 設定打卡地點

1. 點擊「新增地點」
2. 填寫地點資訊：
   - 地點名稱（如：總公司、分公司）
   - GPS 座標
   - 允許打卡半徑（公尺）
3. 可設定 WiFi 輔助驗證

#### WiFi 輔助驗證

每個打卡地點可設定允許的 WiFi 名稱：
1. 啟用「WiFi 輔助驗證」
2. 輸入允許的 WiFi SSID（每行一個）
3. 可選擇「僅 WiFi 驗證」模式

### 打卡時間限制

**路徑：** 「系統設定」→「打卡時間限制」

1. 啟用/停用時間限制
2. 設定限制時段（如 23:00 - 05:00）
3. 自訂限制時段提示訊息

### 加班費計算設定

**路徑：** 「系統設定」→「加班費計算設定」

設定各類加班費率：
- 平日加班費率
- 休息日加班費率
- 國定假日加班費率

### 薪資相關設定

**路徑：** 「系統設定」

| 項目 | 說明 |
|-----|------|
| 勞保設定 | 勞保費率、上下限 |
| 健保設定 | 健保費率、眷屬人數 |
| 勞退設定 | 勞退提撥比例 |
| 獎金設定 | 年終獎金計算方式 |

## 報表匯出

### 考勤報表

**路徑：** 「報表」→「考勤報表」

1. 選擇報表期間（年/月）
2. 可選擇部門篩選
3. 點擊「產生報表」
4. 預覽後可匯出：
   - PDF 格式
   - CSV 格式（可用 Excel 開啟）

### 薪資報表

**路徑：** 「報表」→「薪資報表」

1. 選擇報表月份
2. 產生報表
3. 可匯出 PDF 或 CSV

### 匯出個人薪資單

1. 在薪資列表點擊「預覽」
2. 點擊「列印薪資單」
3. 可儲存為 PDF

## 常見操作情境

### 情境 1：新員工入職

1. 新增員工基本資料
2. 建立登入帳號
3. 設定特休天數
4. 通知員工帳號密碼

### 情境 2：員工忘記打卡

1. 收到員工補打卡申請
2. 進入「補登打卡」
3. 補登後通知員工

### 情境 3：批次審核請假

1. 進入「請假審核」
2. 勾選多筆申請
3. 點擊「批次核准」

### 情境 4：月底考勤結算

1. 確認當月所有打卡記錄
2. 處理未結的異常記錄
3. 匯出月度考勤報表
4. 核對加班和請假記錄
5. 進行薪資計算

### 情境 5：員工離職

1. 結算補休餘額
2. 處理年假結算
3. 停用員工帳號
4. 更新員工狀態
