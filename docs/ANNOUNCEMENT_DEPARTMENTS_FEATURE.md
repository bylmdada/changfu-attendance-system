## 使用說明

### 管理員發布公告流程：
1. 點擊「申請公告」按鈕
2. 填寫標題、內容、優先級等基本信息
3. 選擇公告對象：
   - **全部部門**：選擇此選項將向所有員工發送通告
   - **指定部門**：選擇此選項並勾選特定部門
4. 完成其他設定後點擊「發布公告」或「儲存草稿」

### 公告對象顯示：
- **全部部門**：顯示為藍色「全部部門（通告）」標籤
- **指定部門**：顯示「指定部門：部門1、部門2...」

## 技術細節

### 數據存儲
- `isGlobalAnnouncement`: Boolean - 標記是否為全員通告
- `targetDepartments`: String (JSON) - 存儲目標部門列表，全員通告時為 null

### 部門常數
使用 `/src/constants/departments.ts` 中的 `DEPARTMENT_OPTIONS` 確保部門列表一致性。

### 驗證邏輯
- 前端：表單提交前檢查部門選擇
- 後端：API 接收數據時驗證部門設定合理性
