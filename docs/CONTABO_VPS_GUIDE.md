## 選擇理由

### 儲存類型：SSD（非 NVMe）

| 類型 | 速度 | 價格 | 建議 |
|------|------|------|------|
| HDD | 慢 | 便宜 | ❌ |
| **SSD** | 快 | 中等 | ✅ 推薦 |
| NVMe | 極快 | 較貴 | ⚠️ 非必要 |

**原因**：40 人使用只需 SSD 效能的 0.01%，NVMe 是過度消費。

### 作業系統：Ubuntu 22.04 LTS

| 系統 | 建議 |
|------|------|
| Ubuntu 22.04 LTS | ✅ 最穩定 |
| Ubuntu 24.04 LTS | ⚠️ 太新 |
| Debian 12 | ✅ 也可 |
| CentOS | ❌ 停止維護 |

### 機房位置

| 機房 | 到台灣延遲 | 建議 |
|------|-----------|------|
| **Singapore** | 50-80 ms | ✅ 推薦 |
| **Japan** | 40-70 ms | ✅ 推薦 |
| Germany | 250-300 ms | ❌ |
| USA | 150-200 ms | ❌ |

## 購買步驟

1. 前往 [Contabo](https://contabo.com/)
2. 選擇 **Cloud VPS** → **VPS S (SSD)**
3. 設定：
   - Region: **Singapore** 或 **Japan**
   - Storage Type: **SSD**
   - Operating System: **Ubuntu 22.04 (64 bit)**
4. 購買（支援信用卡/PayPal）
