# 家庭照顧假管理指南

## 📋 家庭照顧假概述

家庭照顧假是為了讓員工能夠照顧家庭成員而設立的請假類型，適用於需要照顧生病、受傷或其他需要照護的家庭成員的情況。

## 🔄 家庭照顧假規則

### 基本規則
- **請假類型代碼**: `FAMILY_CARE`
- **請假類型名稱**: 家庭照顧假
- **年度上限**: 7天
- **薪資給付**: 不給薪
- **申請條件**: 需照顧家庭成員（配偶、子女、父母等）

### 適用情況
1. 家庭成員生病需要照護
2. 家庭成員發生意外需要陪伴
3. 家庭成員有重要醫療程序需要陪同
4. 其他家庭緊急照護需求

### 申請流程
1. **提前申請**: 建議提前3天申請（緊急情況可事後補申請）
2. **填寫申請單**: 詳細說明請假事由和照護對象
3. **主管審核**: 由直屬主管進行審核
4. **HR確認**: HR部門確認年度額度使用情況

## 💰 薪資計算

### 家庭照顧假薪資規則
```typescript
// 家庭照顧假薪資計算 (不給薪)
const familyCareLeavePay = {
  dailyPay: 0,           // 日薪: 0元 (不給薪)
  hourlyPay: 0,          // 時薪: 0元 (不給薪) 
  isPaid: false,         // 是否給薪: 否
  deductFromSalary: true // 從月薪中扣除: 是
};

// 月薪扣除計算
function calculateFamilyCareSalaryDeduction(
  baseSalary: number,
  workingDaysInMonth: number,
  familyCareDaysTaken: number
): number {
  const dailySalary = baseSalary / workingDaysInMonth;
  return dailySalary * familyCareDaysTaken;
}
```

### 薪資扣除範例
- **月薪**: 50,000元
- **當月工作日**: 22天  
- **家庭照顧假**: 2天
- **日薪**: 50,000 ÷ 22 = 2,273元
- **扣除金額**: 2,273 × 2 = 4,546元
- **實發薪資**: 50,000 - 4,546 = 45,454元

## 📊 額度管理

### 年度額度追蹤
```sql
-- 查詢員工家庭照顧假使用情況
SELECT 
  e.employeeId,
  e.name,
  COUNT(lr.id) as total_requests,
  SUM(lr.totalDays) as days_used,
  (7 - COALESCE(SUM(lr.totalDays), 0)) as remaining_days
FROM employees e
LEFT JOIN leave_requests lr ON e.id = lr.employeeId 
  AND lr.leaveType = 'FAMILY_CARE'
  AND lr.status = 'APPROVED'
  AND YEAR(lr.startDate) = YEAR(CURDATE())
GROUP BY e.id, e.employeeId, e.name
ORDER BY e.employeeId;
```

### 額度警告機制
- **剩餘1天**: 系統提示警告訊息
- **額度用完**: 禁止再次申請
- **超額申請**: 自動拒絕申請

## 🔍 審核要點

### 主管審核檢查項目
1. ✅ **申請事由合理性**: 確認是否符合家庭照顧假適用情況
2. ✅ **年度額度確認**: 檢查是否超過7天年度上限
3. ✅ **工作安排**: 確認請假期間工作交接安排
4. ✅ **證明文件**: 必要時要求提供相關證明文件

### HR審核檢查項目  
1. ✅ **政策合規性**: 確認符合公司政策和勞動法規
2. ✅ **額度計算**: 確認年度額度使用正確
3. ✅ **薪資影響**: 確認薪資扣除計算正確
4. ✅ **記錄維護**: 更新員工請假記錄

## 📋 常見問題 FAQ

### Q1: 家庭照顧假是否需要證明文件？
**A**: 一般情況下建議提供，緊急情況可事後補件。包括醫院診斷書、緊急狀況證明等。

### Q2: 家庭照顧假可以按小時申請嗎？
**A**: 可以，支援半日假和小時假，但仍需計入7天年度額度。

### Q3: 家庭照顧假是否影響年終考核？
**A**: 不影響。家庭照顧假是法定假別，不應影響員工考核評等。

### Q4: 超過7天額度怎麼辦？
**A**: 可申請事假或其他假別，但需符合各假別規定。

### Q5: 年度額度是否可以累積到下一年？
**A**: 否，家庭照顧假額度不可累積，每年重新計算。
