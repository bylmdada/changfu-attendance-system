# GPS 定位權限問題解決指南

## 問題說明
當系統出現 "GPS定位權限被拒絕" 錯誤時，表示瀏覽器無法存取您的位置資訊。

## 解決方法

### Chrome 瀏覽器
1. **允許位置存取**：
   - 在網址列左側找到 🔒 或 ℹ️ 圖示
   - 點擊後選擇「網站設定」
   - 將「位置」設定改為「允許」
   - 重新整理頁面

2. **檢查瀏覽器設定**：
   - 進入 Chrome 設定 → 隱私權和安全性 → 網站設定
   - 點擊「位置」
   - 確認該網站在「允許」清單中

### Safari 瀏覽器
1. **網站權限設定**：
   - 在網址列點擊 「aA」 圖示
   - 選擇「網站設定」
   - 將「位置」改為「詢問」或「允許」

2. **系統設定檢查**：
   - iOS：設定 → 隱私權與安全性 → 定位服務
   - macOS：系統偏好設定 → 安全性與隱私權 → 位置服務

### Firefox 瀏覽器
1. **網站權限**：
   - 點擊網址列左側的盾牌圖示
   - 找到「位置」權限並設定為「允許」
   - 重新整理頁面

### Edge 瀏覽器
1. **網站權限**：
   - 點擊網址列左側的鎖頭圖示
   - 將「位置」權限改為「允許」
   - 重新整理頁面

## 系統設定檢查

### Windows 系統
1. 確認系統定位服務已開啟：
   - 設定 → 隱私權 → 位置
   - 開啟「位置服務」
   - 允許應用程式存取位置

### macOS 系統
1. 系統偏好設定 → 安全性與隱私權 → 隱私權
2. 選擇「位置服務」
3. 確認已勾選並允許瀏覽器存取

### Android 系統
1. 設定 → 位置 → 確認位置服務已開啟
2. 在瀏覽器 App 權限中允許位置存取

### iOS 系統
1. 設定 → 隱私權與安全性 → 定位服務
2. 確認定位服務已開啟
3. 在 Safari 設定中允許位置存取

## 常見問題

### Q: 為什麼需要GPS定位？
A: 系統使用GPS確保員工在指定地點打卡，維護考勤制度的公平性。

### Q: GPS不準確怎麼辦？
A: 請到戶外或靠近窗戶的位置，等待GPS訊號穩定後再進行打卡。

### Q: 可以關閉GPS打卡嗎？
A: 管理員可在系統設定中調整GPS打卡規則，一般員工無法自行關閉。

## 技術支援
如果按照上述步驟仍無法解決問題，請聯繫系統管理員或IT支援團隊。
