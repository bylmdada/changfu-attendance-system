# 加班管理模組說明

## 功能概述
長福會考勤系統的加班管理模組提供完整的加班申請、審核和管理功能。

## 加班規則
1. **最小時間單位**：0.5小時（30分鐘）
2. **最大加班時數**：每日不超過4小時  
3. **工時計算**：工作滿8小時後才開始計算加班
4. **總工時限制**：一天總工作時間不能超過12小時（正常8小時 + 加班4小時）
5. **加班時段**：必須在17:00之後開始（假設正常工作時間為09:00-17:00）

## 使用流程

### 員工申請加班
1. 進入系統儀表板
2. 點選「加班管理」
3. 點擊「申請加班」按鈕
4. 填寫加班申請表單：
   - 加班日期
   - 開始時間（須在17:00後）
   - 結束時間
   - 申請原因
   - 工作內容（選填）
5. 提交申請，等待主管審核

### 主管審核加班
1. 管理員或HR進入「加班管理」頁面
2. 查看待審核的加班申請
3. 點擊「批准」或「拒絕」按鈕
4. 系統自動記錄審核結果和時間

### 查看加班記錄
- 員工可查看自己的加班申請記錄和狀態
- 管理員可查看所有員工的加班記錄
- 支援按狀態、日期範圍篩選
- 支援按員工姓名或原因搜索

## API 端點
- `GET /api/overtime-requests` - 獲取加班申請列表
- `POST /api/overtime-requests` - 提交加班申請
- `PATCH /api/overtime-requests/[id]` - 審核加班申請
- `DELETE /api/overtime-requests/[id]` - 刪除加班申請

## 權限控制
- **員工**：可申請加班、查看自己的申請記錄、刪除自己未審核的申請
- **管理員/HR**：可審核所有加班申請、查看所有記錄、刪除任何待審核申請

## 系統驗證
- 開始時間必須在17:00之後
- 時數自動以0.5小時為單位進位
- 同一天不能重複申請加班
- 只能刪除待審核狀態的申請
- 已審核的申請無法修改或刪除

## 注意事項
- 系統假設正常工作時間為8小時（09:00-17:00）
- 加班時數會影響薪資計算
- 管理員可在薪資管理中查看加班時數統計
- 建議員工提前申請，避免當天臨時申請
