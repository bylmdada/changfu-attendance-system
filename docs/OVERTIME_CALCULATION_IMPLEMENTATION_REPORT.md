### 🧮 計算規則說明

#### 平日加班
```
前2小時：平日每小時工資額 × 4/3 (約1.34倍)
2小時後：平日每小時工資額 × 5/3 (約1.67倍)
```

#### 休息日加班
```
工作時間計費規則：
- 2小時以內 → 按4小時計給
- 2-8小時 → 按8小時計給  
- 8-12小時 → 按12小時計給

費率：
- 前8小時：× 4/3倍
- 8小時後：× 5/3倍
```

#### 國定假日加班
```
全日雙倍薪：平日每小時工資額 × 2
```

#### 例假日加班
```
條件：僅天災、事變或突發事件
費率：全日雙倍薪 + 需補假
```

### 📊 計算範例

**假設月薪 48,000 元的員工：**
- 平日每小時工資額：NT$ 200 (48,000 ÷ 240)
- 平日加班3小時：NT$ 1,002 (2小時 × 268 + 1小時 × 334)
- 休息日加班6小時：NT$ 2,144 (按8小時計，8小時 × 268)
- 國定假日加班6小時：NT$ 2,400 (6小時 × 400)

### 📝 使用指南

#### 基本使用
```typescript
import { calculateOvertime, OvertimeType } from '@/lib/overtime-calculator';

// 計算平日加班費
const result = calculateOvertime(OvertimeType.WEEKDAY, 3, 48000);
console.log(result.overtimePay); // 加班費總額
console.log(result.details); // 詳細計算過程
```

#### 綜合薪資計算
```typescript
import { calculateMonthlyPayroll } from '@/lib/payroll-calculator';

const payroll = calculateMonthlyPayroll(employee, attendanceRecords, year, month);
console.log(payroll.grossPay); // 總薪資
console.log(payroll.netPay); // 實領薪資
```
