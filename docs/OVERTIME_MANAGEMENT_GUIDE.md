# 加班管理規範

## 加班申請流程

```
1. 事前口頭報備 → 2. 主管同意 → 3. 提出加班申請 → 4. 系統審核
```

### 流程說明

| 步驟 | 負責人 | 說明 |
|------|--------|------|
| 1. 口頭報備 | 員工 | 加班前須向主管報備 |
| 2. 口頭同意 | 主管 | 主管同意後員工方可加班 |
| 3. 加班申請 | 員工 | 於系統提出加班申請 |
| 4. 系統審核 | 主管/HR | 審核通過後計入薪資 |

> [!IMPORTANT]
> 加班須**事前口頭報備**並獲得主管同意。

---

## 超時下班處理

當員工下班打卡時間超過班表時間：

```
超時下班打卡
    │
    ├─ 「非公務因素」→ 記錄原因 → 結束
    │
    └─ 「公務因素」→ 記錄原因 → [選填] 提交加班申請
```

### 公務因素加班申請（選填）

選擇「公務因素」後，系統可顯示加班申請表單：

| 欄位 | 預填值 |
|------|--------|
| 加班開始時間 | 班表下班時間 |
| 加班結束時間 | 實際打卡時間 |
| 加班事由 | 空白（必填） |

## 加班時間計算

### 重要原則

| 來源 | 計算內容 |
|------|----------|
| **打卡記錄** | 僅記錄工時，**不計算加班** |
| **加班申請** | 需審核通過，才計入加班費 |

### 未來規劃

- **最小計費單位**：可配置（預設 30 分鐘，可設為 1 分鐘）
- **計算方式**：精確計算（無進位）
