# 靈活薪資條系統使用指南

## 系統概述

我們已經為薪資系統增加了靈活性，以更好地應對未來薪資加薪和保險費調整的需求。

## 主要功能

### 1. 動態薪資項目管理
- 支持動態添加/刪除薪資項目
- 靈活的項目分類和排序
- 配置驅動的項目管理

### 2. 增強的薪資條
- 動態顯示收入和扣除項目
- 支持項目描述和數量
- 響應式設計適配移動端

## 管理員功能

### 薪資項目配置
1. 訪問 `/payroll/config` 頁面
2. 查看所有已配置的薪資項目
3. 新增項目：
   - 輸入項目代碼（如 'PERFORMANCE_BONUS'）
   - 輸入項目名稱（如 '績效獎金'）
   - 選擇類型（收入/扣除）
   - 選擇分類（薪資/津貼/保險/稅務/退休金/其他扣除）
   - 設置排序順序
4. 編輯現有項目
5. 停用不需要的項目

### 常見薪資項目配置

| 代碼 | 名稱 | 類型 | 分類 |
|------|------|------|------|
| BASE_SALARY | 基本薪資 | 收入 | 薪資 |
| OVERTIME_PAY | 加班費 | 收入 | 薪資 |
| BONUS | 獎金 | 收入 | 津貼 |
| LABOR_INSURANCE | 勞工保險 | 扣除 | 保險 |
| HEALTH_INSURANCE | 健康保險 | 扣除 | 保險 |
| INCOME_TAX | 所得稅 | 扣除 | 稅務 |

## 員工功能

### 查看薪資條
1. 訪問薪資查詢頁面
2. 選擇年月查看薪資記錄
3. 點擊 "HTML薪資條" 或 "列印薪資條" 按鈕
4. 薪資條將動態顯示所有配置的項目

## 系統優勢

### 適應變化
- **薪資調整**: 輕鬆適應基本薪資變化
- **保險費調整**: 動態更新勞保、健保費率
- **新項目**: 快速添加獎金、津貼等新項目

### 維護便利
- **集中管理**: 所有項目配置集中管理
- **即時生效**: 配置變更立即應用到新生成的薪資條
- **版本追蹤**: 配置變更可追蹤

## 技術實現

### 數據結構
- `PayrollItemConfig`: 項目配置表
- `PayrollItem`: 具體薪資項目記錄
- 動態關聯支持靈活的項目組合

### API 接口
- `/api/payroll/config`: 項目配置管理
- `/api/payroll/payslip`: 增強的薪資條生成
- 支持動態項目查詢和顯示

## 遷移說明

系統已自動遷移現有薪資記錄到新格式，無需手動操作。

## 故障排除

### 薪資條顯示問題
1. 檢查項目配置是否正確設置
2. 確認用戶權限是否正確
3. 查看瀏覽器控制台是否有錯誤信息

### 配置管理問題
1. 確保用戶具有管理員權限
2. 檢查項目代碼是否唯一
3. 確認分類選擇正確

## 聯繫支持

如有問題，請聯繫系統管理員或查看技術文檔 `docs/flexible-payroll-system.md`。
