## ✅ 薪資條管理系統功能特色

### 🏗️ 核心架構
- **多標籤介面設計**：範本管理、系統設定、預覽功能三大模組
- **響應式設計**：支援桌面與行動裝置操作
- **即時預覽**：模擬薪資條產生與顯示效果

### 📋 範本管理功能
- **完整範本 CRUD**：新增、編輯、刪除、檢視薪資條範本
- **預設範本系統**：內建標準薪資條格式，可設定多個範本
- **範本狀態管理**：啟用/停用範本，預設範本保護機制
- **自定義內容**：公司資訊、員工欄位、收入/扣除項目設定

### ⚙️ 系統設定功能
1. **自動產生設定**
   - 排程產生：每月指定日期與時間自動產生
   - 彈性配置：可設定每月 1-28 日的任何時間

2. **發送配置**
   - 多種方式：電子郵件、列印、或兩者並行
   - 範本化郵件：支援變數替換（員工姓名、月份等）
   - 客製化內容：自定義郵件主旨與內容

3. **保存管理**
   - 保存期限：設定薪資條保存月數（12-120個月）
   - 歸檔機制：自動歸檔舊資料（6-60個月）

4. **安全設定**
   - PDF 密碼保護：確保薪資條安全性
   - 員工同意機制：電子薪資條接收確認

### 🖥️ 預覽功能
- **即時預覽**：基於實際範本配置的薪資條預覽
- **模擬資料**：使用真實格式的測試資料展示
- **格式驗證**：確保範本設定的正確性
- **PDF 下載**：支援薪資條 PDF 檔案生成

### 資料結構設計
```typescript
interface PayslipTemplate {
  // 基本資訊
  id, name, description, isDefault, isActive
  
  // 頁首設定
  headerConfig: 公司名稱、地址、Logo設定
  
  // 員工區塊
  employeeSection: 員工ID、部門、職位等顯示控制
  
  // 收入項目
  earningsSection: 基本薪資、加班費、獎金等設定
  
  // 扣除項目  
  deductionsSection: 勞健保、所得稅等設定
  
  // 匯總區塊
  summarySection: 總收入、總扣除、實發薪資
  
  // 頁尾設定
  footerConfig: 產生日期、簽名、自定義文字
  
  // 格式設定
  formatting: 字體、頁面大小、邊距等
}

interface PayslipSettings {
  // 自動產生
  autoGeneration: 啟用狀態、排程日期時間
  
  // 發送配置
  distribution: 發送方式、郵件範本
  
  // 保存設定
  retention: 保存期限、歸檔規則
  
  // 安全設定
  security: 密碼保護、員工同意
}
```
