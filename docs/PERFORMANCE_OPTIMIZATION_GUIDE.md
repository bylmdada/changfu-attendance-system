## 🔧 故障排除

### **如果資料庫仍然慢：**
1. **重啟開發伺服器**
   ```bash
   # 停止服務 (Ctrl+C)
   # 重新啟動
   npm run dev:https-network
   ```

2. **手動執行索引優化**
   ```bash
   sqlite3 prisma/dev.db < database-optimize.sql
   ```

3. **清理資料庫**
   ```bash
   npm run prisma:reset
   npm run prisma:seed
   ```
