# 按比例獎金發放使用指南

## 功能概述

按比例獎金功能是針對到職未滿一年員工的獎金計算與發放系統，確保新進員工能按其實際服務期間長度獲得公平的獎金待遇。系統自動計算服務月數，並根據公司政策按比例發放年終獎金和三節獎金。

## 核心特色

### 智慧服務期間計算
- **精確月數計算**: 考慮入職日期的具體日期，精確計算到天的服務月數
- **跨年度處理**: 正確處理跨年度的服務期間計算
- **不足月處理**: 對不足整月的服務期間按日數比例計算

### 獎金類型支援
1. **年終獎金**: 通常等同於一個月基本薪資，按年度服務月數比例發放
2. **三節獎金**: 固定金額（如5,000元），按服務期間比例發放
   - 春節獎金（2月發放）
   - 端午節獎金（6月發放）  
   - 中秋節獎金（9月發放）

### 資格條件設定
- **最低服務期限**: 可設定不同獎金類型的最低服務月數要求
- **在職狀態檢查**: 確保只有在職員工才能領取獎金
- **比例發放門檻**: 可設定服務多久後不再按比例計算（如滿12個月後全額發放）

## 計算邏輯說明

### 年終獎金計算
```
年終獎金 = 基本薪資 × (服務月數 ÷ 12個月)

條件:
- 最低服務期間: 3個月
- 計算基準期間: 當年度1月1日至12月31日
- 服務期間起算: 取「到職日期」與「年度開始日」之較晚者
```

### 三節獎金計算  
```
三節獎金 = 固定金額 × (服務月數 ÷ 12個月)

條件:
- 最低服務期間: 1個月
- 計算基準日期: 各節慶月份的最後一天
- 服務滿12個月後: 不再按比例，直接給予全額
```

### 實際計算範例

#### 範例1: 年終獎金計算
**員工資訊:**
- 姓名: 王小明
- 到職日期: 2024年3月15日
- 基本薪資: NT$ 50,000

**計算過程:**
1. 服務期間: 2024/03/15 → 2024/12/31
2. 服務月數: 9.53個月 (精確計算至日)
3. 比例係數: 9.53 ÷ 12 = 79.4%
4. 年終獎金: NT$ 50,000 × 79.4% = NT$ 39,700

#### 範例2: 春節獎金計算
**員工資訊:**
- 姓名: 李小華  
- 到職日期: 2024年1月20日
- 三節獎金標準: NT$ 5,000

**計算過程:**
1. 服務期間: 2024/01/20 → 2024/02/29 (春節獎金計算截止日)
2. 服務月數: 1.33個月
3. 比例係數: 1.33 ÷ 12 = 11.1%
4. 春節獎金: NT$ 5,000 × 11.1% = NT$ 555

## 操作流程

### 步驟 1: 設定計算參數
1. 進入「按比例獎金計算」頁面
2. 設定目標年度
3. 選擇獎金類型（年終獎金或三節獎金）
4. 若選擇三節獎金，需指定具體節慶類型
5. 決定是否自動創建獎金記錄

### 步驟 2: 執行批量計算
1. 點擊「計算獎金」按鈕
2. 系統自動載入所有在職員工
3. 對每位員工進行服務期間與獎金金額計算
4. 顯示計算結果與統計資訊

### 步驟 3: 檢視計算結果
計算完成後，系統將顯示:
- **統計摘要**: 總員工數、符合資格人數、按比例發放人數、獎金總額等
- **詳細清單**: 每位員工的計算明細，包含:
  - 到職日期與服務月數
  - 比例係數與實發金額
  - 資格狀態（符合資格/不符資格/免發放）

### 步驟 4: 選擇發放對象
1. 在計算結果中勾選要發放獎金的員工
2. 可使用「全選」功能快速選擇所有符合資格的員工
3. 系統會自動排除不符資格或金額為0的員工

### 步驟 5: 發放獎金
1. 點擊「發放獎金」按鈕
2. 確認發放人數與獎金總額
3. 系統自動創建獎金記錄
4. 整合補充保費計算
5. 更新員工年度獎金累計

## 系統設定說明

### 獎金政策配置
管理員可在後台設定以下參數:

**年終獎金設定:**
```json
{
  "bonusType": "YEAR_END",
  "eligibilityRules": {
    "minimumServiceMonths": 3,     // 最低服務3個月
    "mustBeActive": true,          // 必須為在職狀態
    "proRatedForPartialYear": true // 啟用按比例計算
  },
  "calculationFormula": "base_salary * service_months / 12"
}
```

**三節獎金設定:**
```json
{
  "bonusType": "FESTIVAL", 
  "eligibilityRules": {
    "minimumServiceMonths": 1,     // 最低服務1個月
    "mustBeActive": true,          // 必須為在職狀態
    "proRatedForPartialYear": true,// 啟用按比例計算
    "proRatedThreshold": 12        // 服務滿12個月後全額發放
  },
  "defaultAmount": 5000,           // 標準金額NT$ 5,000
  "calculationFormula": "fixed_amount * service_ratio"
}
```

## 報表功能

### 綜合獎金報表
系統提供完整的年度獎金發放報表，包含:

1. **年終獎金統計**
   - 符合資格人數與比例發放人數
   - 獎金總額與平均金額
   - 按部門的分布統計

2. **三節獎金統計**
   - 各節慶的發放統計
   - 按比例發放的詳細分析
   - 新進員工獎金趨勢

3. **員工明細報表**
   - 個別員工的所有獎金記錄
   - 服務期間與比例計算明細
   - 補充保費計算資訊

### 報表匯出
- **Excel格式**: 支援複雜的數據分析與圖表製作
- **PDF格式**: 適合正式文件與存檔
- **會計系統整合**: 提供標準格式供會計系統匯入

## 常見問題解答

### Q1: 員工在年中離職，如何計算年終獎金？
**A**: 系統會計算該員工在當年度的實際服務月數，按比例給予年終獎金。離職日期會作為計算截止日。

### Q2: 員工跨年度入職，如何計算第二年的年終獎金？
**A**: 系統以每年1月1日為計算起點，只計算當年度的服務期間。例如前年12月入職的員工，隔年年終獎金會按當年度服務的12個月計算。

### Q3: 三節獎金是否需要按各節慶分別計算？
**A**: 是的，每個節慶的獎金都是獨立計算的。春節獎金按到該年2月的服務月數計算，端午節獎金按到6月的服務月數計算，以此類推。

### Q4: 如果員工基本薪資在年中調整，年終獎金如何計算？
**A**: 系統使用計算當下的基本薪資作為年終獎金的基準。建議在薪資調整時同步更新員工資料。

### Q5: 按比例計算的獎金是否需要扣取補充保費？
**A**: 是的，所有獎金都需要納入補充保費計算。系統會自動整合獎金金額到年度累計中，並計算相應的補充保費。

### Q6: 可否手動調整計算結果？
**A**: 系統提供的計算結果可在獎金記錄建立後進行調整。建議先使用自動計算功能，再根據特殊情況進行個別調整。

## 注意事項

### 操作建議
1. **定期檢查員工資料**: 確保到職日期、基本薪資等資訊正確
2. **年終前批量處理**: 建議在12月中旬進行年終獎金的計算與確認
3. **節慶前預先處理**: 各節慶前1-2週完成獎金計算與發放作業
4. **保留計算記錄**: 所有計算過程與調整記錄都會被完整保存

### 權限管理
- **人事主管**: 具備完整的計算與發放權限
- **部門主管**: 可查看所屬部門的計算結果
- **一般員工**: 只能查看個人的獎金記錄

### 合規考量
- 按比例發放政策應符合勞動法規
- 獎金發放應有明確的公司政策依據
- 建議定期檢視政策合理性與公平性
