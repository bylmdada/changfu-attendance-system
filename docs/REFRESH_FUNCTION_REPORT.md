### 🎛️ **刷新功能詳細說明**

#### **手動刷新操作流程：**
1. 點擊 "🔄 刷新" 按鈕
2. 按鈕變為 "🔄 刷新中..." (禁用狀態)
3. 發送 API 請求到 `/api/system-maintenance?action=overview`
4. 更新系統健康、維護任務、問題列表等數據
5. 更新頁面刷新時間戳
6. 恢復按鈕為可用狀態

#### **自動刷新特性：**
- **靜默更新** - 不干擾用戶操作
- **智能間隔** - 30秒平衡及時性與性能
- **錯誤容忍** - 單次失敗不影響後續自動刷新

#### **視覺反饋：**
- 🔄 **旋轉圖示** - 載入時動畫效果
- ⏰ **時間戳** - 顯示最後刷新時間
- 🚫 **禁用狀態** - 防止重複操作
- ✅ **成功提示** - 數據更新確認
