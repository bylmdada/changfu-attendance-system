1. **Prisma 類型錯誤**
   ```bash
   rm -rf node_modules/.prisma
   npx prisma generate
   ```

2. **Next.js 編譯錯誤**
   ```bash
   rm -rf .next
   npm run dev
   ```

3. **Port 佔用錯誤**
   ```bash
   pkill -f "npm run dev"
   npm run dev
   ```

## 📊 系統架構更新

### 部門權限矩陣
每個部門可設定的權限類型：
- 🗓️ 請假申請審核權限
- ⏰ 加班申請審核權限  
- 🔄 調班申請審核權限

### 權限管理流程
1. **管理員設定** → 選擇員工 → 選擇權限類型 → 選擇負責部門
2. **權限生效** → 被授權員工可在對應頁面看到負責部門申請
3. **審核操作** → 批准/拒絕申請並記錄審核資訊
