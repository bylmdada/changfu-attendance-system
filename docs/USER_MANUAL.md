## 系統登入

### 登入方式

1. 開啟瀏覽器，輸入公司考勤系統網址
2. 輸入您的帳號（員工編號）
3. 輸入密碼
4. 點擊「登入」按鈕

### 首次登入

- 首次登入時會要求您變更預設密碼
- 密碼需符合以下規則：
  - 至少 8 個字元
  - 包含大小寫字母
  - 包含數字

### 忘記密碼

請聯繫人事部門重設密碼。

## 首頁功能

登入後會進入首頁，顯示以下資訊：

| 區塊 | 說明 |
|-----|------|
| 今日打卡狀態 | 顯示今日上下班打卡時間 |

| 待處理事項 | 待審核的請假或加班申請 |
| 快速功能 | 常用功能快捷入口 |

## 快速打卡

### 打卡方式

1. 點擊首頁的「快速打卡」按鈕
2. 系統會自動獲取您的 GPS 位置
3. 確認位置正確後，點擊「確認打卡」

### GPS 打卡注意事項

- 請確保手機已開啟定位服務
- 需在公司設定的打卡範圍內（通常 35 公尺內）
- 如定位失敗，請稍等片刻重試

### 打卡時間限制

- 夜間時段（23:00 - 05:00）可能暫停打卡服務
- 具體限制時段請依公司規定

## 考勤記錄查詢

### 查詢步驟

1. 點擊選單中的「考勤記錄」
2. 選擇要查詢的月份
3. 查看每日的打卡記錄

### 記錄說明

| 狀態 | 說明 |
|-----|------|
| ✅ 正常 | 準時上下班 |
| ⚠️ 遲到 | 上班打卡超過規定時間 |
| ⚠️ 早退 | 下班打卡早於規定時間 |
| ❌ 缺勤 | 當日未打卡 |
| 📝 請假 | 已請假 |

## 請假申請

### 申請步驟

1. 點擊「請假申請」
2. 選擇請假類型（特休、病假、事假等）
3. 選擇請假日期和時間
4. 填寫請假事由
5. 點擊「送出申請」

### 請假類型

| 類型 | 說明 |
|-----|------|
| 特休假 | 年度特別休假 |
| 病假 | 需附診斷證明（視公司規定）|
| 事假 | 需提前申請 |
| 婚假 | 結婚可請 |
| 喪假 | 依親屬關係不同天數 |
| 補休 | 使用加班補休時數 |

### 審核流程

1. 送出申請後，主管會收到通知
2. 主管審核後您會收到結果通知
3. 可在「我的申請」查看審核狀態

## 加班申請

### 申請步驟

1. 點擊「加班申請」
2. 選擇加班日期
3. 輸入預計加班時段
4. 選擇補償方式（補休或加班費）
5. 填寫加班理由
6. 點擊「送出申請」

### 補償方式

| 方式 | 說明 |
|-----|------|
| 補休 | 加班時數轉換為補休時數 |
| 加班費 | 依勞基法計算加班費 |

### 注意事項

- 請在加班前提出申請
- 加班需經主管同意
- 單日加班不得超過 4 小時

---

## 補休使用

### 查詢補休餘額

1. 進入「我的假期」
2. 查看「補休餘額」欄位

### 申請補休

1. 點擊「請假申請」
2. 請假類型選擇「補休」
3. 選擇日期和時間
4. 送出申請

### 補休期限

- 補休有使用期限，請注意到期日
- 到期未使用將依公司規定結算

## 常見問題

### Q1: GPS 定位失敗怎麼辦？

**A:** 請確認：
- 手機定位服務已開啟
- 已授權瀏覽器使用定位
- 在戶外或窗邊定位較準確

### Q2: 忘記打卡怎麼辦？

**A:** 請聯繫主管或人事部門協助補登記錄。

### Q3: 請假申請被駁回怎麼辦？

**A:** 請與主管確認駁回原因，修正後重新申請。

### Q4: 加班時數何時會入帳？

**A:** 加班經審核通過後，補休時數會立即入帳；加班費則於次月薪資發放。

### Q5: 密碼過期或被鎖定？

**A:** 請聯繫人事部門重設密碼。
